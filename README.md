# telegram-sales-manager
# Telegram-бот менеджер по продажам (логистика из Китая)

Бот для обработки обращений клиентов, консультаций по услугам и сбора заявок.

## Функции
- Приветствие и описание услуг компании
- Многошаговая форма сбора заявки (имя, телефон, услуга, детали)
- Автоматическая передача заявки админу в личку
- Живой диалог с клиентом

## Технологии
- Python
- aiogram 3.x
